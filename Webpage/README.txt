THE ARCHITECT - How to Play
================================

OPTION 1 (Easiest):
- Double-click start.html
- If nothing happens, try Firefox (works best)
- If using Chrome, you may need to allow local files

OPTION 2 (If Option 1 fails):
- Double-click run.bat (included)
- Follow the instructions


In order to make this playable (without extra work), all code is put into a single file!
